# Crypto Arbitrage Scanner Telegram Bot

Telegram-бот, который сканирует **спотовые рынки** 10 популярных централизованных бирж и присылает уведомления о монетах, где наблюдается заметная разница в цене (потенциально прибыльные арбитражные связки).

**Важно**: это **не автоматический торговый бот**. Бот только **показывает** связки. Покупку и продажу нужно совершать **вручную**.

## Поддерживаемые биржи (на февраль 2026)

- Bybit
- MEXC
- Gate.io
- HTX (Huobi)
- BitMart
- KuCoin
- OKX
- CoinEx
- Poloniex
- BingX

## Основные возможности

- Регистрация по инвайт-коду
- Платная подписка (кол-во дней)
- Настройки пользователя:
  - объём сделки в USDT
  - выбор стратегии (Минимальный риск / Сбалансированная / Максимум прибыли)
  - выбор активных бирж (минимум 2)
- Уведомления о связках с расчётом:
  - спреда (%)
  - чистой прибыли после примерных комиссий (1% + $1)
  - примерного времени жизни окна
- Фильтрация по минимальному спреду и прибыли в зависимости от выбранной стратегии
- Ограничение по количеству сообщений в 5 минут на пользователя
- Периодическая проверка каждые ~5 минут
- Простое меню и управление через кнопки
- Админ-функции (создание инвайт-кодов)

## Как это работает (очень упрощённо)

1. Каждые ~5 минут бот запрашивает текущие цены всех USDT-пар на 10 биржах
2. Группирует цены по монете
3. Для каждого пользователя смотрит только выбранные им биржи
4. Находит минимальную и максимальную цену
5. Считает спред и примерную чистую прибыль
6. Если связка проходит фильтры стратегии → отправляет сообщение с ссылками на торговые страницы

## Риски и важные предупреждения

**Использование бота НЕ гарантирует заработок и может привести к убыткам.**

Самые частые причины потери денег при попытке арбитража:

- **Исчезновение окна** за секунды (самая частая причина)
- **Огромные спреды на низколиквидных монетах** → невозможно купить/продать нужный объём
- **Слиппедж** и **спайк** цены во время перевода
- **Задержки ввода/вывода** на биржах (иногда часы)
- **Комиссии ввода-вывода** (часто съедают всю прибыль)
- **Изменение баланса** на биржах во время перевода
- **Блокировка аккаунта** биржей за подозрительную активность
- **Фишинговые** сообщения, поддельные боты, кража API-ключей

**Никогда** не используйте средства, которые не готовы потерять полностью.

## Установка и запуск (для разработчиков / энтузиастов)

### Требования

- Python 3.9+
- pip install -r requirements.txt
- (рекомендуется) виртуальное окружение

### Шаги

1. Скопируйте проект

```bash
git clone https://github.com/lev4ikysss/Arbitrage
cd https://github.com/lev4ikysss/Arbitrage
```

2. Установите зависимости

   ```bash
   pip install -r requirements.txt
   ```
3. Создайте и заполните config.conf

   ```ini
   [tg]
   token = 123456:AAF1b2C3d4...

   [database]
   path = data/bot.db

   [codes]
   path = data/codes.json

   [bot]
   payment_warning = 7
   ```

   4. (опционально) отредактируйте начальные настройки в codes.json и создайте первый админ-код
   5. Запустите

      ```bash
      python main.py
      ```

      ### Структура проекта


      ```bash
      .
      ├── config.conf
      ├── data/
      │   ├── codes.json
      │   └── listeners.json
      ├── main.py               # основная логика бота
      ├── requirements.txt
      ├── utils/
      │   ├── birges.py         # запросы к API бирж
      │   ├── database.py       # sqlite + настройки пользователей
      │   └── settings.py       # чтение конфига
      └── LICENSE               # Unlicense (общественное достояние)
      ```

   ## Лицензия

   Unlicense — общественное достояние. Делайте с кодом что хотите.


## Рекомендации перед реальным использованием

* протестируйте на маленьких суммах ($10–50)
* проверьте комиссии на ввод/вывод на всех биржах
* используйте **только лимитные ордера**
* никогда не переводите деньги сразу после получения сигнала
* ведите статистику (сколько связок реально закрываются в плюс)

Удачи. И помните:  **большинство арбитражных окон на споте уже давно съедаются высокочастотниками** .
